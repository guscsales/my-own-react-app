# my-own-react-app

Aplicação React do zero para o conteúdo de Frontend Intelligence da FIAP.

## Comandos

Para executar, execute os comandos abaixo.

```bash
npm install
npm start
```

Para gerar um build de produção, execute o comando abaixo.

```bash
npm build
```
